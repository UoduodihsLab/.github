### 🛠 我们在做什么 (What We Do)

我们热衷于探索复杂系统的边界，从底层硬件到高并发分布式服务：

* **Distributed Systems:** 致力于构建低延迟、高可用的分布式判题系统 (OJ) 与节点管理方案。
* **Edge Computing:** 探索 MQTT 协议在边缘端与机器人（Wheeled Robots）上的应用。
* **Infrastructure:** 深度定制 PVE 虚拟化环境、LXC 容器化部署及 2.5G/10G 家庭骨干网方案。
* **Retro-Tech & DIY:** 维护老旧硬件（CRT/Legacy Handsets）并将其集成至现代 IoT 生态。
